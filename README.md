# Django Custom Admin

Projeto desenvolvido com o objetivo de **estudar a customização da interface administrativa do Django**, explorando a alteração visual do painel padrão e a integração com temas modernos como **AdminLTE**.

O Django possui uma das interfaces administrativas mais poderosas entre frameworks web, permitindo gerenciar dados rapidamente a partir dos modelos da aplicação. Essa interface é gerada automaticamente com base nos modelos definidos no projeto. :contentReference

Neste projeto, o foco foi compreender **como personalizar o Django Admin**, modificando aparência, estrutura e comportamento da área administrativa.

---

# 🎯 Objetivo do projeto

Estudar e implementar:

- Customização visual do **Django Admin**
- Integração com **AdminLTE**
- Organização do painel administrativo
- Ajustes de layout e estrutura
- Preparação de uma base para **dashboards administrativos mais modernos**

O **AdminLTE** é um template open source baseado em **Bootstrap**, amplamente utilizado para construção de **painéis administrativos e dashboards**, oferecendo diversos componentes prontos como menus, gráficos e tabelas. 

---

# 🧰 Tecnologias utilizadas

- Python
- Django
